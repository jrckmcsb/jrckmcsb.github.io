What is Mantis Bug Tracker

MantisBT is an open source issue tracker that provides a delicate balance between simplicity and power. Users are able to get started in minutes and start managing their projects while collaborating with their teammates and clients effectively. Simply Mantis Bug Tracker is a open source tool for bug tracking
